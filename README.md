# FlowBoard

FlowBoard is a modern visual workflow and whiteboard application built to help users create, organize, and manage interactive flows with an intuitive drag-and-drop interface.

---

## ✨ Features

- 🎨 Interactive flow-based canvas
- 🧩 Drag-and-drop node system
- ⚡ Fast and responsive UI
- 🌙 Modern clean interface
- 🔗 Connect and manage workflows visually
- 💾 Persistent state management
- 📱 Responsive design

---

## 🚀 Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS / CSS Modules
- **State Management:** Zustand / Context API
- **Backend (Optional):** Node.js / Express
- **Database (Optional):** PostgreSQL + Drizzle ORM
- **Package Manager:** npm / Bun
